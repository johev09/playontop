What's this?

It's a desktop app, which lets you watch your or local videos and keep it always on top of all other windows. So now instead of missing any scene or stopping your work to watch a video, you can carry on with your work while watching your videos.

How to install?

  1. Install Windows Desktop App
    Allow permission to firewall (when asked) to allow extension to work
  2. Install Chrome Extension
  3. That's All Folks!
